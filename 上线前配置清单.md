## ⚠️ 需你在微信公众平台（后台）操作——代码无法替代

### 1. 配置《隐私保护指引》（必须，否则隐私弹窗不生效 / 审核被拒）
- 路径：**微信公众平台 → 设置 → 服务内容 → 用户隐私保护指引**
- 填写并**发布**指引，在「收集个人信息」中声明用途：
  - 头像、昵称（个人中心登录）
  - 相机（扫码识别商品条码/二维码）
- 发布后 `wx.getPrivacySetting` 才会返回 `needAuthorization: true`，弹窗才会在首次进入时弹出。

### 2. 申请订阅消息真实模板 ID（到期提醒推送）
- 路径：**微信公众平台 → 功能 → 订阅消息 → 我的模板 → 选用**
- 选择「保质期到期提醒 / 食品过期提醒」类**一次性**模板，获取模板 ID
- 用真实 ID **替换** `utils/subscribe.js` 第 4 行：
  ```js
  const TEMPLATE_IDS = ['真实模板ID_1', '真实模板ID_2'];
  ```
- 注意：一次性订阅需用户**每次授权**，不承诺永久提醒（文档 11.2）。

### 3. ICP 备案（必须，备案通过才能正式发布/被搜索到）
- 路径：**微信公众平台 → 设置 → 小程序备案**
- 个人主体可备案，按流程提交资料，等待管局审核。

### 4. 服务类目
- 路径：**微信公众平台 → 设置 → 基本设置 → 服务类目**
- 建议选 **「工具 → 效率」** 或 **「生活服务 → 美食」**。

### 5. 清空设备示例数据（发布前必须）
- 开发者工具 → **清缓存 → 全部清除**；或在小程序「我的 → 清空所有数据」
- 确保上传的代码包 / 你的测试机不含测试食品，否则审核看到的是你的演示数据。
